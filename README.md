# Attendance Queue System with Geofencing

A complete web-based attendance system with automatic queue management and 100-meter geofencing.

## Features
- ✅ Unique numbers starting from 1
- ✅ Automatic queue when within 100m of office
- ✅ Real-time notifications
- ✅ No crowding - members wait outside until called
- ✅ Mobile-friendly interface
- ✅ Works on any smartphone
